# OSVi Biomechanics Database — Demo Prototype

A local web app for querying the OSVi patient biomechanics database with an AI-powered chatbot.

## Getting Started

**1. Navigate to the prototype folder**
```
cd prototype
```

**2. Install dependencies**
```
pip install -r requirements.txt
```

**3. Set your API key (PowerShell)**
```
$env:ANTHROPIC_API_KEY = "your-api-key-here"
```

**4. Run the app**
```
python app.py
```

Then open **http://localhost:5050** in your browser.
